# FTC Programming Training: Slides & Resources

## Overview

This directory contains comprehensive programming training materials for FIRST Tech Challenge teams, starting from zero Java knowledge up to FTC SDK proficiency.

**Target Audience**: Complete beginners in Java and FTC programming
**Duration**: 4 weeks (2 weeks Java foundations + 2 weeks FTC SDK)
**Format**: Presentation slides + hands-on exercises + reference materials

---

## Contents

### 1. Week 1-2: Java Foundations
**File**: `Week1-2_Java_Foundations.md`

Learn the fundamentals of Java programming before diving into FTC-specific code.

#### Topics Covered:
- Classes and Objects
- Variables and Data Types
- Operators and Math
- Conditionals (if/else)
- Loops (for, while)
- Methods and Functions
- Arrays and Collections
- String Manipulation
- Comments and Code Style

#### Learning Outcomes:
- Understand object-oriented programming concepts
- Write simple Java programs with functions and loops
- Use different data types appropriately
- Make decisions in code with conditionals

#### Exercises:
14 hands-on exercises ranging from basic variable assignment to implementing arrays

---

### 2. Week 3-4: FTC SDK Basics
**File**: `Week3-4_FTC_SDK_Basics.md`

Master the FTC Software Development Kit to control actual robots.

#### Topics Covered:
- OpMode Structure and Lifecycle
- Hardware Mapping
- Motor Control
- Servo Control
- Gamepad Input Handling
- Telemetry and Debugging
- TeleOp Programs (driver-controlled)
- Autonomous Programs (self-running)

#### Learning Outcomes:
- Understand how FTC programs execute
- Connect Java code to robot hardware
- Write your first TeleOp (driver-controlled) program
- Write your first Autonomous program
- Debug code using telemetry

#### Exercises:
5 hands-on exercises: motor control, servo control, gamepad handling, autonomous movement

---

### 3. Quick Reference Guide
**File**: `Quick_Reference_Guide.md`

A one-stop reference for common Java and FTC SDK patterns.

#### Contents:
- Java Syntax Quick Reference
- FTC SDK Quick Reference
- Complete step-by-step example from setup to first OpMode
- Debugging Checklist
- Code Snippets Library (copy-paste ready)
- Best Practices
- Common Mistakes and Fixes
- Next Topics to Explore

#### Purpose:
Use this as a quick lookup when you need to remember syntax or patterns. Much faster than searching!

## How to Use These Materials

### For Self-Study:
1. **Start with Week 1-2**: Read each slide, run the code examples yourself
2. **Do the exercises**: Type in the code, test it, break it, fix it
3. **Move to Week 3-4**: Apply Java knowledge to actual FTC hardware
4. **Use Quick Reference**: Keep it open while coding as a reference

### For Teaching/Group Study:
1. **Project slides**: Use Week 1-2 slides on a display
2. **Code along**: Have students type examples in real-time
3. **Do exercises together**: Discuss solutions and different approaches
4. **Transition to FTC**: Move to Week 3-4 once comfortable with Java
5. **Build something**: Have students design and code their own OpMode

### For Competition Preparation:
1. Ensure all team members complete Weeks 1-4
2. Assign specific topics based on roles:
   - **Mechanical engineers**: Focus on motor/servo control
   - **Autonomous programmers**: Study autonomous patterns
   - **Driver coaches**: Understand gamepad input handling
3. Use Quick Reference for on-the-fly troubleshooting

## Common Questions

### Q: How long does this take?
**A**: Week 1-2 typically takes 10-15 hours if you do all exercises. Week 3-4 takes 8-12 hours. Total: 18-27 hours for complete beginner.

### Q: Do I need to memorize all the syntax?
**A**: No! Use the Quick Reference Guide as a cheat sheet. Professional programmers look things up constantly.

### Q: Can I skip Java and go straight to FTC?
**A**: Not recommended. Java fundamentals are essential to understand FTC SDK. You'll be lost without understanding classes, methods, loops, etc.

### Q: What if I'm already a programmer?
**A**: Skim through Week 1-2 and focus on Week 3-4 FTC-specific concepts.

### Q: Can I use these slides in my team?
**A**: Yes! These are free to use for educational purposes. Share with your team, other teams, schools, etc.

## Troubleshooting Materials

### Slide won't render properly
- Markdown files should open in any text editor
- Use a Markdown viewer for better formatting (VS Code has built-in support)
- GitHub also renders Markdown automatically

### Code examples don't work
- Check Java/Android Studio version compatibility
- Verify all imports are correct
- See Debugging Checklist in Quick Reference Guide
- Check FTC Slack/Discord for help

### Exercises are too hard
- Go back and review previous slides
- Look at Quick Reference snippets
- Try a simpler version first
- Don't be discouraged - programming takes practice!
